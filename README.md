# SortAlgorithm
Sort algorithm implemented by Erlang. They are:
  1.Bubble sort;
  2.Insert sort;
  3.Select sort;
  4.Shell sort;
  5.Heap sort;
  6.Merge sort;
  7.Quick sort.
